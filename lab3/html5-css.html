HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet. É a quinta versão da linguagem HTML.
header 😀 coloquei no cabeçalho
nav 😀 coloquei no menu de navegação com links para as páginas
main 😀 Coloquei no conteúdo principal sempre
article 🙄 não usei, mas indica porção do documento que pode ser vista de forma independente
section 😀 usado bastantes vezes (Para dividir conteúdo de Multimédia, Fotografias ou Video) para dividir/especificar uma parte do conteúdo
aside 🙄 Não usado, mas indica anotações, publicidade ou outros assuntos desconexos.
footer 😀 Usado em todas as páginas em rodapé (colocado no canto inferior direito) dentro de um section

CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web.

Tipos de seletores CSS
universal 😀 aplica o estilo a todos os elementos da página, também usado, como * {font-family: 'Syne Mono', monospace;
tipo 😀 usa o nome do marcador a estilizar e também usado várias vezes h1 ou h3
classe 😀 associa um nome a um estilo, e usado como por exemplo no footer (class="footerClass")
id 😀 permite identificar univocamente um elemento do HTML, usado por exemplo na linha 17 do ficheiro info e em tantos outros sítios
atributo 🙄 Não usado
pseudo-classe 😀 foram usados todos eles ,link (antes de ser clicado), visited (depois de clicado.), hover (quando o rato passa) ou active (quando o elemento é clicado) -> usados no elemento a
pseudo-elemento 😀 Usado no h1,h3
Combinadores de seletores
Seletores descendentes 🙄 Não usado
Seletores filhos 🙄 Não usado
Seletores adjacentes 😀 usado em todos os p que estão colocados depois de h3
Seletor imediatamente adjacente 🙄 não usado
Agrupar os seletores,separando-os por virgulas 😀 usado em h1,h2,legend
